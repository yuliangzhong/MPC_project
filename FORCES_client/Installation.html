Welcome to FORCESPRO!

Installation

To install your client, after you've unzipped it, add the client folder to your Matlab or Python path.
Then, you can start using it by running a script from the Examples folder of your FORCESPRO client.

To use the code generator on Matlab you need to setup your mex compiler by running mex -setup on your Matlab command line.

Supported Software

FORCESPRO supports the following compilers:

  • On Windows platforms:
    • Microsoft Visual C++ compilers (compilers provided with Visual Studio)
    • MinGW
    • Windows SDK 7.1
  • On Linux platforms: GCC
    • GCC
  • On Mac platforms:
    • Clang (Xcode)
    • GCC

Supported Matlab versions: 2016a and newer.
You can check in the following link which of the above compilers are supported on each Matlab version: Matlab Supported Compilers

You can check each version of Matlab by changing the url depending on the version: 2018a → YYYYx
i.e. SystemRequirements-ReleaseYYYYx_SupportedCompilers.pdf

Supported Python versions:

  • 2.7 (only low-level interface)
  • 3.4 (only low-level interface)
  • 3.6
  • 3.7
  • 3.8
